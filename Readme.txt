# Alien Invasion Web

A web-based version of the Alien Invasion game using Flask and Pygame.

## Description

This project is an adaptation of the classic Alien Invasion game into a web application. It utilizes Flask for the web framework and Pygame for the game logic.

## Getting Started

### Prerequisites

- Python (version 3.9.7)
- Flask
- Pygame

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Sathvika-16/alien-invasion.git

After you clone the repository: run app.py on the terminal 

Usage
Navigate to the provided URL, and you will be able to play the Alien Invasion game directly in your web browser.
